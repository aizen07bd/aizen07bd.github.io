1단계에서 배운 변수, 데이터 타입, 함수, 제어 흐름(if, match, loop)을 모두 활용한 '숫자 맞히기 퀴즈 게임'을 만들어 보겠습니다.
이 코드를 직접 작성해 보고 실행 결과를 블로그에 올리면 1단계 학습 인증으로 완벽할 거예요!

## 🦀 숫자 맞히기 게임 코드 (src/main.rs)

```rust
use std::io; // 입출력을 위한 표준 라이브러리 가져오기
fn main() {
    println!("--- 숫자 맞히기 게임 ---");

    let secret_number = 7; // 정답 숫자 (나중에 랜덤으로 바꿀 수 있어요)

    loop {
        println!("1부터 10 사이의 숫자를 입력하세요:");

        // 1. 사용자 입력 받기
        let mut guess = String::new();
        io::stdin()
            .read_line(&mut guess)
            .expect("입력을 읽지 못했습니다.");

        // 2. 입력받은 문자열을 숫자로 변환 (Shadowing 활용!)
        // trim()은 공백/엔터 제거, parse()는 숫자로 변환
        let guess: u32 = match guess.trim().parse() {
            Ok(num) => num,
            Err(_) => {
                println!("숫자만 입력해 주세요!");
                continue;
            }
        };

        println!("당신이 입력한 값: {}", guess);

        // 3. 정답 비교 (match 조건문 활용)
        match guess {
            1..=6 => println!("너무 낮아요!"),
            8..=10 => println!("너무 높아요!"),
            7 => {
                println!("축하합니다! 정답입니다. 🎉");
                break; // 정답을 맞히면 루프 종료
            }
            _ => println!("1부터 10 사이의 숫자만 입력해달라니까요?"),
        }
    }
}
```

------------------------------
## 📝 코드 설명 (복습 포인트)

   1. use std::io: 키보드 입력을 받기 위해 외부 모듈을 불러오는 구문입니다.
   2. let mut guess = String::new(): 사용자의 입력을 담을 가변(mut) 문자열 변수를 만듭니다.
   3. Shadowing: guess라는 이름을 문자열에서 숫자(u32) 타입으로 재정의해서 코드를 깔끔하게 유지했습니다.
   4. match와 에러 처리: parse()가 실패할 경우(문자 입력 등) 프로그램이 꺼지지 않게 match로 안전하게 처리했습니다.
   5. loop와 break: 정답을 맞힐 때까지 무한 반복하다가, 맞히는 순간 break로 탈출합니다.

## 💡 숙제 (블로그에 남길 도전 과제)

* 난이도 업: 정답 숫자 7을 고정하지 않고, 매번 바뀌게 하려면 어떻게 해야 할까요? (힌트: rand 크레이트 사용)

## 오류 관련
warning: unused variable: `secret_number` 오류 관련
그 경고는 Rust 컴파일러가 **"변수를 만들어두고 한 번도 사용하지 않았어!"**라고 친절하게 알려주는 것입니다. 프로그램 실행에는 문제가 없지만, 코드를 더 깔끔하게 관리하라는 조언이죠.
제시해 드린 코드의 3단계(정답 비교) 부분에서 match guess 안에 숫자 7을 직접 쓰는 대신, 선언해둔 secret_number 변수를 사용하도록 수정하면 경고가 사라집니다.

* 🛠️ 코드 수정하기
기존의 7 => { ... } 부분을 아래와 같이 변수 이름을 사용하도록 바꿔보세요.
```rust
// ... 생략 ...
// 3. 정답 비교 (변수 secret_number를 활용!)match guess {
    n if n < secret_number => println!("너무 낮아요!"),
    n if n > secret_number => println!("너무 높아요!"),
    n if n == secret_number => {
        println!("축하합니다! 정답입니다. 🎉");
        break; 
    }
    _ => println!("범위를 벗어났어요!"),
}
```
또는 더 간단하게 이렇게도 가능합니다.
```rust
if guess < secret_number {
    println!("너무 낮아요!");
} else if guess > secret_number {
    println!("너무 높아요!");
} else {
    println!("축하합니다! 정답입니다. 🎉");
    break;
}
```
