1단계는 Rust와 친해지는 과정으로, 언어의 문법적인 기초를 다지는 시기입니다. 
다른 언어와 비슷해 보이지만 Rust만의 독특한 규칙들이 섞여 있으니, 이 부분을 중점적으로 살펴보시면 좋습니다.

# 🦀 1단계: 기초 다지기 상세 커리큘럼

## 1. 개발 환경 설정 및 도구 익히기
코드를 짜기 전에 연장부터 챙기는 단계입니다.

* Rustup & Cargo: Rust의 툴체인 관리자인 rustup과 빌드 도구이자 패키지 매니저인 Cargo의 역할을 이해합니다.
* 새 프로젝트 생성: cargo new <프로젝트명> 명령어로 프로젝트 구조(src 폴더, Cargo.toml 파일)를 익힙니다.
* 컴파일과 실행: cargo build, cargo run, cargo check(컴파일 확인만 수행)의 차이를 배웁니다.

## 2. 변수와 가변성 (Immutability)
Rust의 가장 큰 특징 중 하나인 '기본 불변성'을 배웁니다.

* Immutable vs Mutable: 왜 Rust는 변수가 기본적으로 불변(let)인지, 가변성을 주려면 왜 mut 키워드를 써야 하는지 학습합니다.
* 상수(Constants): const와 let의 차이점을 이해합니다.
* Shadowing: 같은 이름의 변수를 다시 선언하여 이전 값을 덮어쓰는 Rust만의 방식을 익힙니다.

## 3. 데이터 타입 (Data Types)
정적 타입 언어로서 Rust가 데이터를 처리하는 방식을 배웁니다.

* 스칼라 타입: 정수형(i32, u32 등), 부동 소수점(f32, f64), 불리언, 문자(char) 타입.
* 복합 타입: 튜플(Tuple)과 배열(Array)의 선언 및 인덱스 접근 방법.
* 타입 추론: 타입을 명시하지 않아도 컴파일러가 알아내는 과정을 이해합니다.

## 4. 함수 (Functions)
코드를 재사용 가능한 단위로 나누는 법을 배웁니다.

* 매개변수와 반환 타입: 함수 선언 시 매개변수와 반환값의 타입을 어떻게 지정하는지 학습합니다.
* 구문(Statements) vs 표현식(Expressions): Rust에서 세미콜론(;) 유무에 따라 반환값이 결정되는 독특한 문법을 익힙니다. (매우 중요!)

## 5. 제어 흐름 (Control Flow)
프로그램의 논리를 구성하는 기본 문법입니다.

* 조건문(if): if가 표현식으로 사용되어 변수에 값을 대입하는 방식을 배웁니다.
* 반복문(loop, while, for): 특히 for 루프에서 범위(Range)를 사용하는 법과 loop에서 값을 반환하는 법을 학습합니다.
